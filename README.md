# Modelo-1er-Parcial
Modelo de primer parcial. Gestion de mercancia
## Descripción
Se solicita implementar un sistema que gestione diferentes tipos de mercancías, clasificadas en
 productos y servicios. Los productos pueden tener descuentos por la compra de una segunda
 unidad, mientras que los servicios pueden estar en promoción. El sistema deberá permitir
 agregar mercancías, calcular precios finales, y consultar los elementos que se encuentran en
 oferta. Se implementan excepciones, polimorfismo, herencia, abstracción y encapsulamiento.
